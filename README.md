## The Problem
A developer made w website, mobile app or even a cli. A developer wants to collect data from users eg. bug report. A developer must create backend for receiving those bug reports or even make a UI to see them. A developer is frustrated 🥴.

## The fix
AnyPost allows developer to create a endpoint that simply can accept any POST request. A developer will only be responsible for creating thier beautifull UI.
AnyPost can accept any body
